# CoffeeShopApp (Exemplo de Padrões de Projeto)

Projeto de demonstração que implementa múltiplos padrões de projeto:
- Factory Method
- Decorator
- Strategy
- Observer
- Command

## Como executar
```bash
python3 main.py
```

## Estrutura
- `factory/` - fábrica de bebidas
- `beverages/` - classes Beverage e Decorators
- `payment/` - strategies de pagamento
- `observer/` - Subject e Observers
- `command/` - encapsula pedidos como comandos

## Arquivos importantes
- `RESUMO.md` - resumo teórico com justificativas
