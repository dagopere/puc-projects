# RESUMO — Estudo de Padrões de Projeto

## Padrões usados
- Factory Method (Criacional)
- Decorator (Estrutural)
- Strategy (Comportamental)
- Observer (Comportamental)
- Command (Comportamental)

## Justificativas resumidas
- Factory Method: organiza criação de bebidas e facilita extensão.
- Decorator: permite montar bebidas dinamicamente (ingredientes).
- Strategy: abstrai métodos de pagamento (pix, cartão, dinheiro).
- Observer: notifica sistemas/usuários quando o pedido muda de estado.
- Command: encapsula execução de pedidos possibilitando enfileiramento/log.
